# YTM Widget 🎵

一個輕量、極簡且無廣告的 YouTube Music 桌面懸浮小工具。
支援背景播放、無縫阻擋廣告，以及一鍵下載高音質單曲功能。

## ✨ 核心特色
- **零廣告干擾**：內建原生事件驅動特務，瞬間跳過所有影音廣告。
- **純粹背景執行**：化身系統列 (Tray) 小工具，乾淨不佔用 Dock / 工作列空間。
- **一鍵下載音樂**：內建 yt-dlp 引擎，輕鬆將當前播放單曲儲存為高音質原聲檔。

---

## 📖 使用說明

1. **啟動與播放**
   點擊小工具面板上的 **「🌐 地球圖示」** 即可開啟 YouTube Music 登入並選擇音樂。選定後，將該網頁視窗關閉或縮小皆不影響背景播放。
2. **一鍵下載**
   點擊面板上的 **「📥 下載圖示」** 即可下載當前播放的音樂。
3. **自訂下載路徑**
   透過右下角系統列 (Tray) 的小工具圖示 ➔ **「點擊右鍵」** ➔ **「⚙️ 設定」**，即可開啟設定介面並選擇您的專屬下載路徑。

## 📥 安裝指南

請至 [Releases](../../releases) 頁面下載符合您作業系統的安裝檔，或參考以下安裝方式。

### 🍎 macOS (Apple Silicon / Intel)

#### 方法一：使用 Homebrew 安裝（🌟 推薦首選）
打開終端機，輸入以下指令即可一鍵完成安裝，並自動繞過系統安全限制：
```bash
brew install --cask kizato1018/tap/ytm-widget
```

#### 方法二：手動下載 DMG 安裝
1. 下載 `.dmg` 檔案，打開後將 `ytm-widget.app` 拖曳至 **應用程式 (Applications)** 資料夾。
2. **【重要】解除系統安全限制**
   由於本軟體為開源免費發布（未經 Apple 付費開發者簽章），macOS 會預設阻擋執行並顯示「檔案已損壞」。
   請打開 **終端機 (Terminal)**，複製貼上以下指令並按下 Enter（執行時需輸入 Mac 開機密碼）：
   
   ```bash
   sudo xattr -rd com.apple.quarantine "/Applications/ytm-widget.app"
   ```
   
3. 執行完畢後，即可至應用程式中正常點擊開啟。

### 🪟 Windows
至 Releases 頁面下載 `.msi` 或 `.exe` 檔案，雙擊執行並依照提示即可完成安裝。

### 🐧 Linux
- **.AppImage (免安裝版)**：下載後，右鍵點擊檔案 ➔ 內容 ➔ 權限，勾選「允許作為程式執行」（或在終端機執行 `chmod +x 檔案名.AppImage`），接著雙擊即可開啟。
- **.deb (安裝版)**：適用於 Ubuntu/Debian 系統。下載後雙擊透過軟體中心安裝，或在終端機輸入以下指令：
  
  ```bash
  sudo dpkg -i 檔案名.deb
  ```
